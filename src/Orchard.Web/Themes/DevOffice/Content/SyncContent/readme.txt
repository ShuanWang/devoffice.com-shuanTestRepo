
Note: These instructions are for syncing content on a fresh install

To sync Images:
1. Copy all the folders in "/Themes/DevOffice/Content/SynContent/Media" folder to "/Themes/Media/Default" folder manully (not through CMS)


To Import Content XML file:
1. Go to Modules, search for Import Export module and enable the module.
2. After enabling the module, click on Import/Export from left menu.
3. Click browse and upload OfficeDevStagingContent.xml file. Then, click import which will create all the images and content items.
4. This does not include any widgets or its content like footer. This can be used to sync only images and content items.
